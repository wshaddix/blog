# NoSQL Databases

Non-relational databases optimized for specific data models and access patterns.

## Types of NoSQL Databases

| Type | Example | Best For |
|------|---------|----------|
| Document | MongoDB, CosmosDB | Flexible schemas, JSON data |
| Key-Value | Redis | Caching, sessions |
| Column-Family | Cassandra | Time-series, high write volume |
| Graph | Neo4j | Relationships, social networks |
