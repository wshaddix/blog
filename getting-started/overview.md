# Getting Started

Welcome! This site is a comprehensive guide to software engineering - from fundamentals to leadership.

## Who This Is For

- **Aspiring engineers** starting their journey
- **Junior developers** building foundational skills
- **Senior engineers** looking to level up or transition to management
- **Engineering managers** refining their craft
- **Anyone** wanting to understand how software gets built

## How to Use This Site

### If You're Brand New to Programming

1. Start with [Prerequisites](/getting-started/prerequisites)
2. Set up your [Workstation](/getting-started/workstation-setup)
3. Follow the [Learning Path](/getting-started/learning-path)

### If You're an Experienced Developer

Jump directly to topics that interest you:
- [Advanced C#](/csharp/advanced-features/generics) for language mastery
- [Design Patterns](/patterns/overview) for proven solutions
- [Building a SaaS](/building-a-saas/overview) for a comprehensive project

### If You're Moving Into Management

Start with [Career Development](/career-development/overview):
- Understand the [Career Ladder](/career-development/career-ladder)
- Learn [People Management](/career-development/people-management/one-on-ones) skills
- Explore [Team Operations](/career-development/team-operations/skill-matrices)
