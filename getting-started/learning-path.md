# Suggested Learning Path

A roadmap through the material based on your experience level.

## Path 1: New to Software Engineering

**Duration:** 6-12 months (part-time)

### Phase 1: Foundations (Months 1-2)
1. [Prerequisites](/getting-started/prerequisites) - Verify you're ready
2. [Workstation Setup](/getting-started/workstation-setup) - Get your tools
3. [Fundamentals: Big O](/fundamentals/algorithms/big-o-notation) - Start thinking about complexity
4. [Fundamentals: Data Structures](/fundamentals/data-structures/arrays) - Arrays through trees
5. [Fundamentals: OOP Principles](/fundamentals/programming-concepts/oop-principles) - Foundation for C#

### Phase 2: C# Proficiency (Months 3-4)
1. [C# Language Basics](/csharp/language-basics/getting-started) - Full section
2. [C# Collections](/csharp/language-basics/collections) - Practice with data structures
3. [C# Advanced: LINQ](/csharp/advanced-features/linq) - Essential for productivity
4. [C# Advanced: Async](/csharp/advanced-features/async) - Modern C# requirement

### Phase 3: Building Things (Months 5-6)
1. [Databases: SQL Fundamentals](/databases/relational/sql-fundamentals) - Data persistence
2. [Entity Framework](/csharp/entity-framework/getting-started) - Database access in C#
3. [ASP.NET Core: Web APIs](/csharp/aspnet-core/web-apis/overview) - Build your first API
4. [DevOps: Version Control](/devops/version-control/git-installation) - Git fundamentals

### Phase 4: Real-World Application (Months 7+)
1. [Building a SaaS](/building-a-saas/overview) - Work through the comprehensive project
2. [Patterns](/patterns/overview) - Reference as needed during the project

## Path 2: Experienced Developer, New to .NET

**Duration:** 2-4 months (part-time)

### Phase 1: C# Quickly (Weeks 1-2)
1. [C# Language Basics](/csharp/language-basics/getting-started) - Skim for syntax differences
2. [C# Advanced Features](/csharp/advanced-features/generics) - Focus on LINQ, async, records

### Phase 2: .NET Ecosystem (Weeks 3-4)
1. [Entity Framework](/csharp/entity-framework/getting-started) - ORM patterns
2. [ASP.NET Core](/csharp/aspnet-core/web-apis/overview) - Web framework
3. [C# Testing](/csharp/testing/unit-testing) - xUnit, Moq

### Phase 3: Apply Knowledge (Weeks 5+)
1. [Building a SaaS](/building-a-saas/overview) - Apply everything
2. [DevOps: Azure](/devops/azure/overview) - Deployment target

## Path 3: Senior Engineer → Management

**Duration:** Ongoing

### Immediate Focus
1. [Career Ladder](/career-development/career-ladder) - Understand the transition
2. [Individual Contributor → Manager](/career-development/career-ladder/individual-contributor) - Specific guidance
3. [One-on-Ones](/career-development/people-management/one-on-ones) - Your most important tool
4. [Giving Feedback](/career-development/people-management/feedback) - Practice immediately

### First 90 Days
1. [Hiring](/career-development/people-management/hiring) - If you're building a team
2. [Team Building](/career-development/people-management/team-building) - Culture matters
3. [Skill Matrices](/career-development/team-operations/skill-matrices) - Know your team

### Ongoing Development
1. [Stakeholder Management](/career-development/soft-skills/stakeholder-management) - Working with others
2. [Technical Writing](/career-development/soft-skills/technical-writing) - Clear communication
3. [Manager → Director](/career-development/career-ladder/director) - Next level

## How to Learn Effectively

1. **Don't just read** - Build things with what you learn
2. **Take notes** - In your own words
3. **Teach others** - Best way to solidify knowledge
4. **Be patient** - Mastery takes time

---

*Pick the path that fits your situation and get started. You can always adjust as you go.*
