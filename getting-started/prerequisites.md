# Prerequisites

What you should know before diving into this material.

## For the Technical Content

### Minimum Requirements

To get the most from the technical sections, you should:

- **Basic programming concepts** - Variables, loops, functions, conditionals
- **Command line basics** - Navigate directories, run commands
- **Text editor familiarity** - VS Code or similar

### Nice to Have

These will accelerate your learning:

- **Any programming language experience** - Python, JavaScript, Java, etc.
- **Basic HTML/web concepts** - How the web works
- **Database basics** - What a database is, simple queries

## For the Management Content

The [Career Development](/career-development/overview) section assumes:

- **Software engineering experience** - You've been an IC
- **Desire to grow** - Either into management or senior IC roles

## What If I'm Missing Prerequisites?

### New to Programming?

1. Take an intro programming course (Python or JavaScript)
2. Build something small - a calculator, todo list
3. Then come back here for C# and software engineering

### Recommended Free Resources

- [freeCodeCamp](https://freecodecamp.org) - Web development basics
- [CS50](https://cs50.harvard.edu) - Harvard's intro CS course
- [The Odin Project](https://theodinproject.com) - Full-stack curriculum

## Self-Assessment

Can you answer these questions?

**Programming Basics:**
- [ ] What's the difference between a variable and a function?
- [ ] What's a loop? What's a conditional?
- [ ] What does it mean to "call" a function?

**Command Line:**
- [ ] How do you list files in a directory?
- [ ] How do you change directories?
- [ ] How do you run a program from the command line?

If you answered yes to most of these, you're ready. If not, spend some time with the prerequisites first - it'll make everything else easier.
